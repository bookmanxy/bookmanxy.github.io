## 前言
当项目启动提示too many connections(超出连接数)，该怎么排查




### 1：查看当前连接情况
```xml
show full processlist;（没有full，则默认100条）
```

### 2：配置的最大连接数
```xml
show variables like '%max_connections%'; 
```

### 3:重新设置最大连接数：
```xml
set global max_connections=1000
```

### 4：查看连接日志：（创建数，连接数、运行数量、缓存数量）
```xml
show status like 'Threads%';
```
